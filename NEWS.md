# lintrhelper 0.1.0

## Initial Release

* Added `create_simple_linter()` for easy XPath-based linter creation
* Added `create_function_call_linter()` for function call linters
* Added `create_assignment_linter()` for assignment operator linters
* Added `test_linter()` for simplified linter testing
* Added `quick_test()` for rapid XPath development
* Added `linter_template()` for code templates
* Added `xpath_patterns()` for XPath reference
* Included example linters: `no_t_f_linter()`, `no_attach_linter()`, `prefer_arrow_assignment_linter()`, `no_sapply_linter()`, `no_one_length_linter()`
* Added comprehensive documentation and vignettes
